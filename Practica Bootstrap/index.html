Fun fact! Sabias que Bootstrap empezó como el "héroe de estilo" dentro de Twitter. Fue creado para salvar a sus herramientas internas de un caos de diseño, y terminó conquistando el mundo web con su superpoder de hacer sitios responsivos y elegantes.

¿Que es Bootstrap?

Bootstrap es un framework front-end que facilita el desarrollo de sitios web responsivos y atractivos, proporcionando componentes predefinidos, un sistema de cuadrícula y herramientas de JavaScript para mejorar la interactividad y el diseño.


Funciones de Bootstrap

Sistema de Cuadrícula

Un sistema de diseño de 12 columnas que permite crear layouts responsivos.

Plugins de JavaScript

Funcionalidades interactivas como menús desplegables, pestañas y herramientas de tooltip.

Clases Utilitarias

Para realizar ajustes rápidos en márgenes, paddings, colores, alineaciones, etc.

Sistema de Rejilla Flexible

Permite crear diseños responsivos usando filas y columnas con clases como .col-sm- y .col-md-.

Personalización con Sass

Permite ajustar variables para modificar estilos globales de manera sencilla.

Ajustes de Visibilidad

Ofrece clases para mostrar u ocultar elementos según el tamaño de la pantalla, como .d-none y .d-sm-block.

Otras Informaciones⤵️

  • Botones: Vienen en varios estilos y tamaños.
  • Modales: Ventanas emergentes para mostrar contenido adicional sin salir de la página.
  • Carruseles: Para mostrar imágenes o contenido en formato de presentación.
  • Alertas: Notificaciones que pueden ser utilizadas para mostrar mensajes importantes.
  • Formularios: Estilos predefinidos para entradas, botones y validación.
  • Navegación: Menús y barras de navegación responsivas y personalizables.

  • Encabezados: Estilos para títulos y subtítulos.
  • Listas: Listas ordenadas y no ordenadas con estilos.
  • Imágenes: Clases para hacer imágenes responsivas.
  • Tablas: Estilos para tablas que facilitan su lectura.

  • Documentación extensa: Proporciona ejemplos claros y una guía completa para cada componente y función.
  • Soporte para accesibilidad: Mejora la usabilidad para personas con discapacidades.
  • Integración con otros frameworks: Funciona bien con tecnologías como Vue, React y Angular.
  • Comunidad activa: Bootstrap tiene una gran comunidad de desarrolladores que contribuyen con temas y extensiones.
